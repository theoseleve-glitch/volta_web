## Pre-launch dress rehearsal

48 hours before launch, run:

```bash
# 1. Theme check
shopify theme check

# 2. Perf audit — home, product, cart
npx lighthouse https://<store>/ --preset=perf --form-factor=mobile
npx lighthouse https://<store>/products/ginger-classic --preset=perf --form-factor=mobile
npx lighthouse https://<store>/cart --preset=perf --form-factor=mobile

# 3. Accessibility
npx @axe-core/cli https://<store>/

# 4. Broken links
npx broken-link-checker https://<store>/ --recursive

# 5. Push to unpublished theme for final QA in admin preview
shopify theme push --unpublished --theme-id=$VOLTA_DEV_THEME_ID

# 6. Promote via Shopify admin (never via CLI)
```
